Créez et gérez facilement des tests de personnalité dynamiques sur votre site WordPress.

== Description ==
Tests Personnalités est une extension WordPress qui vous permet de créer, personnaliser et afficher des tests de personnalité interactifs directement depuis l’interface d’administration. 

- Créez des tests avec plusieurs questions, réponses et règles de résultats via une metabox intuitive.
- Enregistrez et modifiez chaque test comme un contenu personnalisé (Custom Post Type).
- Affichez n’importe quel test sur vos pages ou articles grâce à un simple shortcode : [personality_test id="123"].
- Interface moderne et responsive pour vos visiteurs, avec transitions et UX optimisée.
- Chaque résultat est calculé dynamiquement selon les réponses de l’utilisateur et vos propres critères de scoring.

Idéal pour les sites de coaching, blogs, magazines, ou toute plateforme souhaitant proposer des quiz ou tests de personnalité engageants.

== Installation ==
1. Téléversez le plugin
2. Activez-le
